Place small public-domain or freely licensed GM soundfont here later (e.g. TimGM6mb.sf2 ~6MB) if you want to add FluidSynth bundled playback.

For now, the app uses Windows built-in Microsoft GS Wavetable Synth via rtmidi for zero-install playback.

You can also route to any other installed software synth (VirtualMIDISynth recommended for better quality).
